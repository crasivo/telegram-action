## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a potential security vulnerability in this project, please report it responsibly:

1. **Report:** Send an email to **[crasivodev@gmail.com]** or use GitHub's Private Vulnerability Reporting feature (if enabled in settings).
2. **Details:** Include a description of the vulnerability, steps to reproduce, and a proof of concept if possible.
3. **Response:** You will receive an acknowledgment within 72 hours.

## Best Practices

To keep your credentials safe:

- **Use Secrets:** Always use [GitHub Secrets](https://docs.github.com/en/actions/security-guides/using-secrets-in-github-actions) for `bot_token`.
- **Verify:** Ensure you are using the official version from the Marketplace.

Thank you for helping keep this project secure!
